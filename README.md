# README

## Описание проекта

Этот консольный проект предназначен для фильтрации заказов через командную строку.

## Установка

1. Скачайте репозиторий в виде ZIP-архива или с помощью ссылки на GitHub.
2. Распакуйте архив в удобное для вас место на вашем компьютере.
3. Откройте решение в вашем любимом редакторе кода, который поддерживает проект (например, Visual Studio, JetBrains Rider, VS Code и др.).
4. Запустите решение, скомпилируйте проект, если это необходимо.

## Использование

1. После запуска программы введите команду `help`, чтобы получить список доступных команд и инструкций по их использованию.
2. Используйте доступные команды для выполнения необходимых операций.

## Важно

- Обратите внимание, что удаление файлов, созданных программой, может привести к некорректной работе или сбоям. Пожалуйста, будьте осторожны при удалении данных.

## Поддержка

Если у вас возникли вопросы или проблемы с использованием проекта, не стесняйтесь обращаться за помощью!
